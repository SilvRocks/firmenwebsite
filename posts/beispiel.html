Dieser Beispielbeitrag zeigt, wie einfach Sie neue Inhalte hinzufügen können. Legen Sie einfach eine weitere HTML‑Datei in das Verzeichnis posts/, und dieser Code lädt sie automatisch.

Sie können beliebig viele Beiträge in diesem Ordner ablegen, solange sie auf .html enden. Der Ladevorgang erfolgt clientseitig via JavaScript, deshalb benötigen Sie keinen Server mit dynamischer Logik.
